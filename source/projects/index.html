Page "about" pour GitHub

Ce projet permet de générer facilement une page "à propos" liée à son compte GitHub. Il suffit de forker le projet et de modifier certaines informations pour qu'une page soit disponible directement sur GitHub, par le biais du système des GitHub pages. Plus d'infos en lisant la documentation.

Ruby warrior client

Le site Ruby Warrior est une jolie initiative qui vise l'apprentissage par le jeu. Ici, le jeu consiste à programmer un chevalier qui va se retrouver confronté à tout un tas de difficultés (monstres, pièges, détours ...) pour rejoindre sa princesse. C'est une façon ludique d'apprendre la programmation en Ruby. J'ai passé quelques niveaux mais je n'ai pas (encore ?) terminé le jeu.

Organization Packages

Initialement, ce projet a été développé pour mon entreprise mais j'ai décidé de l'open sourcé pour que d'autres puissent en bénéficier. Il s'agissait d'avoir une vision des dépendances de nos projets. Plus précisément, nous voulions savoir quelles étaient les bibliothèques communes (ou concurrentes) utilisées par nos projets.

Nova et Galactus

Encore en cours de développement, ces 2 projets forment les 2 facettes d'un aggrégateur de flux RSS de type Planet. D'un côté, Galactus, écrit en PHP, récupère les informations des flux RSS ciblés et les exposent sous la forme d'une API. De l'autre, Nova est une interface AngularJS qui interroge l'API pour afficher les informations. 2 planets sont déjà en ligne

  • Planet Jeux de Rôles
  • Planet BD
J'ai encore des choses à peaufiner mais j'ai bon espoir de pouvoir proposer cet aggrégateur à la communauté PHP pour remplacer le planet PHP FR actuel.
